       假设要删除的元素都在这两个List中的第n位置，由于两者都循环查找了n次，省略循环查找这个步骤，直接看删除，由于ArrayList删除元素后，底层数组要往前复制一格，ArrayList底层数组删除元素时间复杂度为Ｏ(n)。再来看LinkedList，LinkedList底层链表删除元素只是简单的修改了一下引用地址，时间复杂度为O(1)。
